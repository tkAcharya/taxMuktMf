# taxMuktMf
taxMuktMf is a privacy-first Chrome Extension designed for Indian investors to automate Mutual Fund Tax Harvesting. Leveraging the Finance Act 2024 rules, it helps you realize up to ₹1.25 Lakh in Long Term Capital Gains (LTCG) with 0% tax liability.  Simply upload your CAS (Consolidated Account Statement) PDF and see the magic happen.
